# Afriqa Store (Next.js + Tailwind)

Kids-first African-inspired fashion store demo (Canada), with basic Shop → Product → Cart → Checkout flow.

## Run
1) Copy `.env.local.example` to `.env.local` (optional for demo admin)
2) Install deps:
   - npm install
3) Start dev server:
   - npm run dev
Open http://localhost:3000

## Notes
- Product data lives in `data/products.json`
- Orders are appended to `data/orders.json` (demo server-side file write).
